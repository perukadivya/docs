# Overview

This documentation covers all data engineering resources for the team.

## Purpose

- Document all data pipelines and their configurations
- Provide reusable SQL and Python code snippets
- Establish coding standards and best practices
- Enable knowledge sharing across the team

## Who Should Use This

- **Data Engineers**: Primary users for pipeline documentation
- **Data Analysts**: SQL reference and query examples
- **Developers**: API documentation and integration guides

## How to Contribute

1. Clone the repository
2. Create a new branch
3. Edit markdown files in your preferred editor
4. Submit a pull request

```{note}
All documentation is written in MyST Markdown, which extends standard Markdown with additional features.
```
